# Personal_Elearning_Platform
Personal Learning platform. A Web application

This project is for personal learning website. 

What this application will do:

- Create a Elearning field. 
eg: CS

- This allows Guests to view learning materials, published by registered users, to be viewed. 


Whats unique :
- Registered users can add their own content in a specific topic. Instead of just viewing the pre-added content.
- Registered users can publish topic.
- Guests can only view topics.
